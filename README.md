# Whisky Hunter (whisky-hunter)

Whisky Hunter is a market research and data platform for whisky collectors, investors, traders, and enthusiasts that aggregates historical auction data from 28 online whisky auction sites into a single database. It tracks trading volumes, winning bids, lot counts, and per-distillery statistics. All trading volumes and winning bids are stated in GBP (£). The Whisky Hunter API provides free, public access to this auction data with no authentication required.

## APIs

### Whisky Hunter API

The Whisky Hunter API provides free, public access to historical whisky auction data aggregated from 28 online whisky auction platforms. Endpoints cover aggregated auction statistics, distillery listings, and per-distillery historical auction data. No authentication is required. All monetary values are in GBP (£). Only lots actually sold (meeting reserve price) are included in trading volume calculations.

- **Human URL:** [https://whiskyhunter.net/api/](https://whiskyhunter.net/api/)
- **Base URL:** `https://whiskyhunter.net/api`
